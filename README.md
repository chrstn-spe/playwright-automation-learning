# Playwright Automation Learning

This repository documents my learning journey in Playwright using TypeScript and JavaScript.

## Objectives

- Learn Playwright fundamentals
- Build maintainable automated UI tests
- Practice assertions and locators
- Organize test suites
- Prepare for QA Automation Engineer roles

---

## Project Structure

### Phase 1

Topics covered:

- Playwright setup
- Basic test execution
- Locators
- Assertions

---

### Phase 2

Topics covered:

- Advanced assertions
- Todo application automation
- Test organization

---

### Phase 3

Topics covered:

- Inventory application automation
- GitHub Actions
- Continuous Integration (CI)

---

## Technologies Used

- Playwright
- TypeScript
- JavaScript
- Node.js
- Git
- GitHub

---

## Running the Project

Install dependencies:

```bash
npm install
```

Run all tests:

```bash
npx playwright test
```

Open the HTML report:

```bash
npx playwright show-report
```
